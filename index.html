📖 使用說明

一、基本操作

  • 新增持股：輸入股票代號、數量、單位（張/股），可選填買入均價以計算總損益。
  • 編輯持股：點擊列表中的「✎」按鈕，可修改股票資訊。
  • 刪除持股：點擊列表中的「✕」按鈕。
  • 分頁管理：支援 1~5 個自訂分頁，可在「進階設定」中調整數量。雙擊分頁名稱可重新命名。
  • 主題切換：點擊右上角圖示切換多種介面主題。

二、API 設定與動態頻率

為了避免超過 API 速率限制，系統會根據您輸入的 Key 數量以及全球股市開盤狀態自動調整頻率：

  • 一組 API Key：正常情況下每 20 秒更新一次。
  • 兩組 API Key（推薦）：正常情況下每 10 秒更新一次，由兩組 Key 輪流請求。
  • 智能休市降頻：系統會自動判斷亞股（08:00-14:00）與美股（21:30-05:00）時段。若所有主要市場皆休市，將自動放緩至每分鐘更新一次，省電又省流量。

三、國際指數報價

除了台股加權指數外，國際股市透過 Cloudflare Worker 代理直接抓取 Yahoo Finance 即時資料，延遲極低。您可以在「進階設定」中勾選欲顯示的國際指數（支援道瓊、那斯達克、S&P 500、費半、日經、韓國）。

四、離線使用與快取

遇到斷網時，系統會自動讀取最後一次的快取報價，並在名稱旁標示 (離線)，待網路恢復後會自動重新連線。
