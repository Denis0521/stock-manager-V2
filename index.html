📖 系統使用說明

一、基本操作與介面

  • 記帳與損益：輸入股票代號、張/股數及買入均價。系統會自動計算目前總現值、今日損益與總未實現損益（ROI）。
  • 資產歷史存檔：每日收盤後，可點擊「存儲今日損益總計」，系統會將當日的資產結算記錄永久存放在手機內，幫您建立長期的資產變化曲線。
  • 佈景主題：點擊右上角「🌙」圖示，即可在極簡科技風、優雅紫羅蘭等多種 3D 擬物化色彩間隨心切換。

二、分頁（組合）的靈活妙用 💡

進階設定中最多可開啟 5 個分頁，雙擊標籤即可重新命名。強烈建議這樣規劃：

  • 家人資產代管：將分頁命名為「我的庫存」、「老婆帳戶」、「小孩教育金」，一台手機就能輕鬆分開管理全家資產、互不干擾。
  • 投資策略分流：建立「長期存股」與「短線波段」分頁，將不同核心邏輯的股票分開追蹤，避免被短期波動影響長期領息心態。
  • 純觀察看盤：建立一個「自選觀察」分頁，只要輸入代號，數量填 0 或不填均價，它就會變成一個極簡純粹的即時報價面板。

三、Fugle API Key 報價引擎原則

為避免觸發免費 API 每分鐘的流量限制，本系統具備智慧防呆與雙金鑰輪替引擎：

  • 單組 Key 模式：盤中正常時每 20 秒 刷新一次；若遇到 API 限制，系統會自動切換至 Yahoo 備援並放慢至 30秒。
  • 雙組 Key 模式（強烈推薦 🌟）：只要填入兩組 Key，更新頻率將直接提速至每 10 秒！系統會完美「左右開弓」輪流調用兩組金鑰，徹底分散限額風險，讓報價如流水般順暢。

四、國際指數報價原則 (Yahoo 備援)

  • 無懼休市斷線：國際大盤（含道瓊、那斯達克、費半及台積電 ADR 等）與台股備援，均採用高效邊緣節點（Cloudflare Worker）介接 Yahoo 報價。
  • 強效防呆快取：當全球市場進入盤後或清晨資料清洗期，系統會自動抓取「昨收價」或「常規市場參考價」補位。即便網路斷線，也會優先顯示手機內最新的暫存檔，確保畫面永遠有數據。

五、慢速本地儲存與備份重要性 ⚠️

  • 絕對隱私：本軟體不使用任何雲端資料庫，您的庫存、損益與歷史紀錄 100% 存放在您手機瀏覽器的「本地儲存空間 (Local Storage)」中。
  • 防止資料遺失（非常重要）：因為資料存在本地，如果您的手機清除了瀏覽器快取、開啟了無痕模式，或是更換了新手機，資料就會歸零。請務必將本網頁「加入主畫面」當作獨立 APP 使用，並養成每週點擊「📥 匯出備份」的習慣，將 JSON 檔妥善保存！

六、強制喚醒技巧

手機系統常會將背景程式休眠。若打開 APP 時報價未即時跳動，只要手指輕觸螢幕滑動或點擊任意空白處，系統內部的「心跳引擎」便會被瞬間喚醒，強迫於背景更新最新報價！
