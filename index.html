📖 系統使用說明

一、主要功能簡介

  • 庫存即時記帳：輸入股票代號、庫存張/股數、買入均價，系統即自動幫您計算現值、今日損益與未實現總損益（ROI）。
  • 資產歷史存儲：收盤後可點擊「存儲今日損益總計」，將當日的資產總額與盈虧永久記錄在手機本地，建立長期資產曲線。
  • 多主題切換：點擊右上角 🌙 顆粒按鈕，可在極簡科技風、擬物化灰灰、優雅紫羅蘭等多種 3D 視覺主題間切換。

二、分頁（組合）的靈活妙用 💡

分頁數量可在進階設定中自訂（1~5 個），雙擊分頁標籤即可重新命名。您可以這樣規劃：

  • 家人資產代管：將分頁命名為「我的庫存」、「老婆帳戶」、「小孩教育基金」，全家人的股票資產在同一台手機就能輕鬆分開管理、互不干擾。
  • 投資策略分流：建立「長期存股（領息）」與「波段短線（價差）」分頁，將不同核心邏輯的股票分開追蹤，避免績效混淆。
  • 市場動態觀察：建立一個「純觀察名單」分頁，只輸入代號、數量填 0 或不填均價，當作純看盤的即時自選股清單。

三、Fugle Key 報價更新原則與建議

為避免觸發 Fugle API 的每分鐘速率限制，系統會根據您的金鑰數量與市場狀態自動進行智慧分流與調頻：

  • 僅填一組 API Key：
    • 盤中正常時段：每 20 秒 刷新一次。
    • 若偶遇網路不穩或 429 限制：自動放緩至 30 秒。
  • 填寫兩組 API Key（推薦 🌟）：
    • 盤中正常時段：更新頻率提速至每 10 秒 一次！系統會採「輪流機制」調用主要與備用 Key，完美分散限額風險。
    • 若偶遇錯誤：自動安全回退至 15 秒。

四、Yahoo Finance 國外指數報價原則

  • 多層容錯防呆：全球國際指數直接透過高效能的邊緣代理網道（Cloudflare Worker）介接 Yahoo 數據。
  • 全時段不化白（—）：新版程式碼內建強大的防呆鏈，就算國際市場遇到盤後或清晨資料清洗，系統也會自動抓取昨收價或常規市場參考價補位，保證畫面不壞死。

五、補充注意事項與日常維護

  • 綠色本地儲存：本軟體不使用任何雲端資料庫，所有庫存與歷史損益均安全地儲存在您手機瀏覽器的本地空間中。
  • 定期匯出備份：當瀏覽器清除快取（或手機重置）時資料可能會遺失。建議每週或在有重大庫存變更後，至進階設定中點擊「📥 匯出備份」下載 json 檔案至電腦或雲端硬碟。
  • 智慧降頻省電：當系統偵測到全球市場皆處於休市時段（如台灣半夜且週末），輪詢計時器會自動放慢至 60 秒，以節省您的手機電力與流量。
